# frontEnd-start

使用了蚂蚁金服的 dumi 框架 [dumi](http://d.umijs.org)

逐步完善自己的前端知识体系，顺便复习基础知识。

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
